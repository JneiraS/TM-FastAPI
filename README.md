## Installation
Pour installer MonProjet, suivez ces étapes :

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/MonProjet.git
   ```
2. Accédez au répertoire du projet :
   ```bash
   cd MonProjet
   ```
3. Installez les dépendances :
   ```bash
   npm install
   ```
